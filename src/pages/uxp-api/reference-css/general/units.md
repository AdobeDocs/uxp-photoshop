# CSS Units

**Since** UXP v3.0

UXP supports several CSS units to make layout easier. Units supported are `em`, `rem`, `vh`, `vw`, `vmin`, `vmax`, `cm`, `mm`, `in`, `pc`, `pt`, `px`.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Values_and_units](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Values_and_units)
