# Attribute selector

**Since** UXP v3.0

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Attribute_selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Attribute_selectors)

**Example**

```css
div[data-row] {
    border: 1px solid red;
}
```
