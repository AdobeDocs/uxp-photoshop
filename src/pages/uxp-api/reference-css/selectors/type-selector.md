# Type selector

**Since** UXP v3.0

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Type_selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Type_selectors)

**Example**

```css
sp-action-button {
    margin-right: 0;
}
```
