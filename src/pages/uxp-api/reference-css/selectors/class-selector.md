# Class selector

**Since** UXP v3.0

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Class_selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/Class_selectors)

**Example**

```css
sp-action-button.icon-only {
    padding: 0;
}
```
