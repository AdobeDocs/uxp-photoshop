# Identifier selector

**Since** UXP v3.0

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/ID_selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/ID_selectors)

**Example**

```css

}
```
