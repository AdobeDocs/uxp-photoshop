# border-bottom-width

**Since** UXP v2.0

Sets the width of the bottom border.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/border-bottom-width](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/border-bottom-width)

## Quirks and Exceptions

* UXP does not support the `content-sizing` box layout model. As such, the width and height of all elements include the padding and border width.
