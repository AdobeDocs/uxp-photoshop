# flex

**Since** UXP v3.0

Shorthand for the `flex-grow`, `flex-shrink`, and `flex-basis` styles.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/flex](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/flex)

**Example**

```css
.someElement {
    flex: 1 0 auto; /* can grow, not shrink, auto basis */
}
```
