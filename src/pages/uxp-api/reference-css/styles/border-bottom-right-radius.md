# border-bottom-right-radius

**Since**  UXP v2.0

Sets the bottom right border radius.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/border-bottom-right-radius](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/border-bottom-right-radius)  

## Quirks and Exceptions

* Not all Spectrum UXP components allow border overrides.
* If you need to clip an image, be sure to set `overflow: hidden`.
