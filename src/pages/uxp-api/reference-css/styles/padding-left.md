# padding-left

**Since**  UXP v2.0

Specifies the left padding for an element.

**See**: [https://developer.mozilla.org/en/docs/Web/CSS/padding-left](https://developer.mozilla.org/en/docs/Web/CSS/padding-left)

**Example**:

```css
.someElement {
    padding-left: 10px;
}
```

## Quirks and Exceptions

* UXP does not support the `content-sizing` box layout model. As such, the width and height of all elements include the padding and border width.
