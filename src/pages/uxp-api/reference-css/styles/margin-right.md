# margin-right

**Since**  UXP v2.0

Specifies the right margin for an element.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/margin-right](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/margin-right)

**Example**:

```css
.someElement {
    margin-right: 1em;
}
```

## Quirks and Exceptions

* Margins do not collapse in UXP.
