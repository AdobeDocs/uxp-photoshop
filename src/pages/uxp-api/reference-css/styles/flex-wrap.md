# flex-wrap

**Since** UXP v3.0

Controls if wrapping is allowed within a flexible container. Supports `nowrap` (default) and `wrap`.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/flex-wrap](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/flex-wrap)

**Example**

```css
.someElement {
    flex-wrap: wrap;
}
```
