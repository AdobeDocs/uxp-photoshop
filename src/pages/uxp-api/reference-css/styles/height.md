# height

**Since** UXP v2.0

Specifies the height for an element.

**See**: [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/height](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/height)

**Example**:

```css
.someElement {
    height: 100vh;
}
```
