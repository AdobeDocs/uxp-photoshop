# ::before

**Since** UXP v3.0

Creates a pseudo-element before the matched element. 

**See:** [https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/::before](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Selectors/::before)

**Example:**

```css
.addBullet > *::before 
    content: "•";
}
```
