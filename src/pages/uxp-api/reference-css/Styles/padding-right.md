# padding-right

**Since**  UXP v2.0

Specifies the right padding for an element.

**See**: [https://developer.mozilla.org/en/docs/Web/CSS/padding-right](https://developer.mozilla.org/en/docs/Web/CSS/padding-right)

**Example**:

```css
.someElement {
    padding-right: 10px;
}
```

## Quirks and Exceptions

* UXP does not support the `content-sizing` box layout model. As such, the width and height of all elements include the padding and border width.
