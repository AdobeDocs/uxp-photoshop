# border-right-width

**Since** UXP v2.0

Sets the width of the right border.

**See**: [https://developer.mozilla.org/en/docs/Web/CSS/border-right-width](https://developer.mozilla.org/en/docs/Web/CSS/border-right-width)

## Quirks and Exceptions

* UXP does not support the `content-sizing` box layout model. As such, the width and height of all elements include the padding and border width.
