# PathPoints

A collection of [PathPoint](/ps_reference/classes/pathpoint/) objects that define a subpath, kept in the [SubPathItem.pathPoints](/ps_reference/classes/subpathitem/#pathpoints) property.

*Added in Photoshop 23.3*
